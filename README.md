Project Overview

This Excel dashboard analyzes customer order data to show sales performance and customer behavior in a simple and visual format.

Features

Top 10 customers analysis

Order status distribution

Total revenue and total products sold

Customer count and referral source analysis

Payment method usage (debit, credit, online, gift card)

Items in cart analysis

VLOOKUP used for automatic data retrieval

Tools Used

Excel Pivot Tables

Pivot Charts

VLOOKUP

KPIs

Purpose

The dashboard helps non-technical users easily understand key business insights and make better decisions.
